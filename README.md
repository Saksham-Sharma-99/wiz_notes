#### ENV VARIABLES

```bash
DJANGO_SECRET_KEY=<super_secret_key>
DJANGO_ENVIRONMENT=<environment>
DATABASE_USER=<user>
DATABASE_PASSWORD=<password>
DATABASE_HOST=<host>
DATABASE_PORT=<port>
DATABASE_NAME=<database_name>
```

to start locally use sample.env file

```bash
cp sample.env .env
```
